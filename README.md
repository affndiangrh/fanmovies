## Getting Started

create `.env.local` file and add your [API key](https://www.themoviedb.org/settings/api)

```env
API_KEY=
```

run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
